## Predictive_Analytics_for_small_Businesses
- this is a data druven decision tool that helps small businesses decide when to buy inventory(restock) based on long term price trends.

- This system analyzes past data to calculate Annual Growth Rate of a commodity.
- It allows users to input their specific restock needs and provides a cleear "Restock" or "Wait" recommendation based on predicted future costs.

FEATURES:
- Trend Based Forecasting: uses past averages to predict price movements
- Interactive system: the user inputs their current stock level and Restock point
- Automated alerts 

HOW TO USE IT
1. Run the main python script
2. Input your current inventory requirements
3. Receive an immediate financial recommendation
